Blog software

18th of March 2019, by OrangePartisan

This blog article has been written to explain how this site, the Realsa Blog, is programmed. This blog is part of the personal / dev demo site of OrangePartisan, programmer at large.

This site has been written using Python 3.7.2, Django 2.1, with Bootstrap 4 for the UI and MySQL as back-end. For this site, Bootstrap is only used to render de UI. All data is handled by Django, and the site does not use jQuery. After POC stage, the blog software was managed by GIT, and published on GitHub here.

At first instance the Django blog app was written by following a tutorial by Traversy Media, which can be found here. The functionality generated with this tutorial was bare-bones and a proof of concept rather then a real blog. All additional software like user authentication, the search function and added features are written by me.

Still, this site is not an empty POC. I have written a few articles, and they can be read on this blog. Although infrequently, new content will be posted here.

{%Realsa

But first a word about Realsa. The total of the personal / dev demo site is written around a fictitious corporate entity named {%Realsa. Realsa stands for Requirements Already Satisfied, een term from the Python programming language. {%Realsa is not a real business, but is being used on this site as a placeholder for the branding of the applications.
